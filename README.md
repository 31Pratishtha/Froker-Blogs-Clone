# Froker Blogs Clone

# How to install

1. Clone the repo
3. Open two separate terminals
4. Navigate to backend directory in one terminal
5. run 'npm i'
6. Setup your mongoDB
7. Run 'npm run start'
8. Navigate to frontend directory in the other terminal
9. run 'npm i'
10. Run 'npm run dev'
